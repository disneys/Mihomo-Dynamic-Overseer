# 🚀 Mihomo Dynamic Overseer

一个基于 JavaScript 的高级 Mihomo (Clash Meta) 配置文件覆写脚本。旨在通过动态逻辑实现节点的自动化分类、智能测速以及特定业务（AI/流媒体/Steam）的精准分流。

## ✨ 核心特性

- **📂 自动化动态分组**：通过正则引擎自动识别全球 16+ 主流国家节点，无需手动干预。
- **🛡️ 零丢弃机制**：引入“其他”分组逻辑，确保任何清奇命名的节点都能被捕获，绝不丢失。
- **🤖 AI 专属优化**：平铺所有可用节点并自动测速，智能剔除不支持 AI 的区域（如香港）。
- **🎮 Steam 免流下载**：内置 Steam 特化规则，默认直连，确保更新游戏不扣代理流量。
- **🚫 广告强力拦截**：集成 `category-ads-all` 规则集，在请求发出前即阻断广告，提升响应速度。
- **🔗 订阅自我保护**：预置订阅域名白名单，即使节点全线崩溃，依然能正常执行订阅更新。
- **🎨 视觉美化**：预设稳定且风格统一的彩色图标集，适配全平台 UI 界面。

## 🛠️ 如何使用

1. 打开你的 Clash 客户端（如 Clash Verge Rev, Mihomo Party 等）。
2. 在 **覆写 (Override)** 或 **脚本 (Script)** 模块中新建一个 JS 脚本。
3. 将仓库中的 `main.js` 内容复制并粘贴。
4. 保存并刷新你的订阅，即可看到美观且智能的节点分组。

## ⚠️ 注意事项

- 必须配合 **Mihomo (Meta) 内核** 使用，以支持 `GEOSITE` 语法。
- 建议开启 `no-resolve` 以保护内网访问隐私。
